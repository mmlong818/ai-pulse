Xiaomi to End MiMo Code Free Access July 26, Moving to Paid Plans

Xiaomi will end MiMo Code's free phase at 6 p.m. Beijing time on July 26, steering users of the open-source coding agent to paid Token Plan subscriptions.

Xiaomi has put an end date on the free run of MiMo Code, its open-source AI coding agent. The tool's complimentary usage phase closes at 18:00 Beijing time (10:00 UTC) on July 26, according to a company notice reported by ITHome on July 23. From that point, developers who want to keep using the bundled Xiaomi models must subscribe to a Token Plan on the MiMo API open platform, and the company is offering a 12 percent discount on first-time subscriptions to ease the transition.

From viral freebie to paid product

MiMo Code launched in mid-June 2026 as a terminal-based coding agent released under the MIT license. Its biggest draw was a built-in, limited-time-free allocation of MiMo-V2.5, the multimodal model Xiaomi says performs on par with Anthropic's Claude Sonnet 4.6; the company has also cited a 62 percent result on SWE-Bench Pro for the tool-and-model combination. That no-registration free tier made the agent an easy trial for developers already juggling subscriptions to rival coding tools.

The monetization switch is narrower than it may sound. The MiMo Code client itself remains open source and model-agnostic: it can connect to DeepSeek, Kimi and GLM models, as well as third-party token plans. What ends on July 26 is free access to Xiaomi's hosted models inside the tool, not the software itself. Xiaomi's Token Plan, sold monthly or yearly, covers the V2.5 model series that powers the agent's default experience.

Why it matters

The move is a bellwether for how quickly AI coding agents are converting hype into revenue. Xiaomi followed the now-standard playbook — open-source the client, give the model away to build habit, then attach a subscription — compressed into roughly six weeks. Whether developers who arrived for a free Claude-class model will pay Xiaomi rather than fall back to DeepSeek or other cheap alternatives will be an early, public test of loyalty in China's crowded coding-agent market, and a signal of how seriously Xiaomi intends to run its model platform as a business.
