HeyGen 推出 companion mode：让 Agent 从"接单"变"导演"

HeyGen 新模式让 AI Agent 先提多个创意方向、中途交storyboard确认再渲染成片，把视频生成从一锤子买卖变成迭代式导演流程。

HeyGen 于 7 月 22 日在 X 上发布 companion mode——一种让 AI Agent 制作视频的新方式，用迭代式的"导演循环"取代一次性生成。启用后，Agent 可以先就一个需求提出多个创意角度，渲染前先交出 storyboard 供确认，再通过多轮评审打磨成片。官方对它的定位是：从"下单买视频"到"亲自导一部片"的差别。

面向 Agent 生态，而非网页端

companion mode 落在 HeyGen 的开放 Agent 工具链里，而不是消费级界面中。它扩展了公司的 HyperFrames 系统——一个专为 Agent 设计、用 HTML 写视频合成的图层——以及其 agent-skills 仓库，后者把数字人创建、视频生成和翻译能力接入 Claude Code、Codex、Cursor 等编码 Agent。近期更新还包括意图层、评审循环，以及通过 BRIEF.md 文件实现的持久用户记忆，让 Agent 能跨项目记住创作者的品牌、语气与偏好。同日 HeyGen 还力推其音频转视频 API：一段语音加一张图片，一次调用即可生成开口说话的视频，还支持同一张图批量生成上百条不同旁白的版本。

这次发布正值"让生成式视频对 Agent 可编程"的行业竞速——同一周，可灵也发布了 MCP 工作流教程。

为什么重要：生成式视频落地的核心障碍是可控性——一次性提示词能出惊艳画面，却难以精确驾驭。storyboard 确认这类结构化检查点直接对症下药；而把它们装进编码 Agent 而非网页 UI，则是对创意工作迁移方向的一次押注。AIGC 厂商正在把自己重新定位为 Agent 生态的基础设施：客户不再只是坐在时间轴前的人，还可以是一个带着需求文档、记忆文件和预算的自主系统。
