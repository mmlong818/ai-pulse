小米 MiMo Code 官宣 7 月 26 日结束限免，转入 Token Plan 付费订阅

小米宣布 MiMo Code 将于 7 月 26 日 18 点结束免费使用阶段，此后需在 MiMo 开放平台订阅 Token Plan 继续使用，首次订阅可享 88 折。

小米旗下 AI 编程工具 MiMo Code 的"免费午餐"进入倒计时。据 IT之家 7 月 23 日报道，小米官方宣布 MiMo Code 将于 7 月 26 日 18 时正式结束免费使用阶段。此后，开发者若想继续使用工具内置的小米模型，需要在小米 MiMo API 开放平台订阅 Token Plan，首次订阅可享 88 折优惠。

六周限免，转身收费

MiMo Code 于 2026 年 6 月中旬发布并以 MIT 协议开源，是一款运行在终端里的 AI 编程 Agent。它最大的卖点是内置限时免费的多模态模型 MiMo-V2.5——小米称其性能对标 Anthropic 的 Claude Sonnet 4.6，并给出了工具加模型组合在 SWE-Bench Pro 上 62% 的成绩。免注册即可上手的免费额度，让它在编程工具订阅费日益高企的当下迅速聚拢了一批尝鲜用户。

需要说明的是，这次收费的范围比字面上要窄：MiMo Code 客户端本身仍然开源，且保持模型中立，可以接入 DeepSeek、Kimi、GLM 等主流模型以及第三方 Token Plan。7 月 26 日之后关闭的，是工具内小米自家托管模型的免费额度，而非软件本身。按月或按年订阅的 Token Plan 覆盖驱动默认体验的 V2.5 系列模型。

为什么重要

这是观察 AI 编程 Agent 商业化速度的一个标志性样本。小米走了一条如今已成标配的路线——客户端开源、模型限免养习惯、再挂订阅收费——但整个周期压缩到了六周左右。冲着"免费的 Claude 级模型"而来的开发者，是留下付费，还是转投 DeepSeek 等低价替代，将成为国内编程 Agent 市场用户粘性的一次公开测验，也能看出小米把模型平台当正经生意经营的决心有多大。
