DeepSeek V4 正式版上线:旧接口今日停服,峰谷计费生效

DeepSeek V4 系列今日转正:deepseek-chat 与 reasoner 旧接口停止服务强制迁移,国内大模型首个 API 峰谷计费同步落地。

DeepSeek 于 7 月 24 日将 V4 系列推至正式版,完成了 6 月底就预告、并写死在 API 文档里的切换:旧版 deepseek-chat 与 deepseek-reasoner 接口今日起停止响应,全部流量迁往 deepseek-v4-pro 与 deepseek-v4-flash。

从预览到量产

V4 系列 4 月 24 日以开源预览版首发——V4-Pro 总参数 1.6 万亿、激活 490 亿,V4-Flash 总参数 2840 亿、激活 130 亿,全系标配 100 万 token 上下文。据 TechNode 报道,正式版将百万上下文固化为全系标准,重点强化智能体任务执行、数学推理与代码生成。国内媒体消息称,正式版经过约两个月工程调优,7 月 21 日前后开启小范围灰度,参与灰测的开发者称其综合表现逼近顶级闭源模型——该说法尚待独立基准测试验证。

新计费与新算力

正式版同步启用 DeepSeek 首个 API 峰谷计费:北京时间 9:00–12:00 与 14:00–18:00 高峰时段价格为平峰两倍,这种用价格调节负载的做法在大模型 API 中尚属罕见。另据国内报道,正式版已完成去 CUDA 化适配,上线首日即兼容多款国产 AI 芯片,与此前 V4 在华为昇腾集群上完成后训练的披露一脉相承。API 继续兼容 OpenAI 与 Anthropic SDK,官方文档还列出了 Claude Code、GitHub Copilot、OpenCode 等编程智能体的接入方式。

为什么重要

DeepSeek 仍是开源前沿模型的标尺,而这次发布的看点不在新能力,而在'工业化':当天硬切换、把推理当电网一样分时定价、国产芯片可迁移——处处透出一家从'研究放卫星'转向'规模化商用'的实验室。对于 Kimi K3、Qwen3.8、MiniMax M3 几天内接连开火的中国模型市场,V4 转正重新划定了所有对手定价与智能体性能都要对照的基准线。
