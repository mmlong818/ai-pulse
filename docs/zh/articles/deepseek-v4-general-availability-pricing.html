DeepSeek V4 正式商用:百万上下文,首创分时计价

DeepSeek V4 于7月19日转入正式商用,Pro 与 Flash 双版本、百万 token 上下文,并首创"峰谷分时"API 定价,低价冲击西方闭源模型。

DeepSeek 的 V4 系列模型于7月19日从预览版转入正式商用(GA),距首次发布约三个月。此次上线包含 Pro 和 Flash 两个版本,均支持一百万 token 上下文窗口;旧版接口将于7月24日切换,留给开发者的迁移时间只有几天。

"电网经济学"进入大模型推理

这次升级最重要的变化不在技术,而在商业模式:DeepSeek 在主流模型 API 中首创"峰谷分时定价"。北京时间上午9点至12点、下午2点至6点为高峰时段,按非高峰价格的两倍计费。非高峰时段,V4 Pro 每百万输出 token 约0.87美元,轻量版 V4 Flash 约0.28美元,缓存命中的输入还有大幅折扣。早期测试者反馈,V4 Pro 在常见任务上的表现已接近 Anthropic 的 Claude Opus 4.8 和 OpenAI 的 GPT-5.6 Sol,而价格只是它们的零头。

密集的国产模型发布潮

V4 转正赶上了中国 AI 最密集的一轮发布周期:月之暗面的 Kimi K3 因需求超出 GPU 供给刚刚暂停新用户注册,阿里 Qwen 团队也在预热下一代旗舰。中国的开源权重与低价模型,正日益成为衡量西方闭源模型定价的全球锚点。

真正值得关注的是定价结构本身。分时计费把电网的峰谷经济学搬进了 AI 推理,等于坦白承认:眼下卡住中国厂商的是 GPU 算力而非模型能力。对西方实验室来说,接近前沿的性能配上白菜价,会持续压缩主流负载的利润空间,把差异化逼向能力金字塔的最顶端。对全球开发者——尤其是欧美之外的市场——V4 的价格让中国模型成为成本敏感应用的理性默认选项,全球 AI 技术栈的分叉正在加深,而这正是两边监管者都在头疼的问题。
