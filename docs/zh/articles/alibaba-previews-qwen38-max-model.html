阿里巴巴预览 2.4 万亿参数旗舰模型 Qwen3.8-Max

阿里 2.4 万亿参数多模态旗舰在 WAIC 期间以预览版亮相，Qwen 团队承诺后续将开放权重。

阿里巴巴 Qwen 团队在上海世界人工智能大会（WAIC）上预览了 Qwen3.8-Max——一款参数规模约 2.4 万亿的混合专家（MoE）模型，官方称其为迄今最强的系统。7 月 19 日的这次发布，距离竞争对手月之暗面推出 Kimi K3 仅隔数日，凸显中国头部实验室的发布节奏正不断加快。

首个万亿级多模态 Qwen 模型

Qwen3.8-Max 是 Qwen 系列中首个参数超过万亿且支持多模态输入的模型，可在单一系统内处理文本、图像、视频与文档理解。阿里表示，预览版在编程、全栈开发、数据分析和办公流程等任务上均超越前代 Qwen3.7-Max。团队在 X 平台的帖文中更进一步，称该模型"仅次于 Fable 5"——即 Anthropic 的旗舰模型。这一大胆说法尚未得到独立基准测试的验证。

预览版已上线阿里的开发者平台，价格约为标准定价的十分之一。这种折扣打法阿里此前就用过，意在正式商用前抢先培育用户。

承诺开放权重

阿里表示后续将开放模型权重，但未给出具体时间表。若在这一规模上兑现，Qwen3.8-Max 将跻身有史以来最大的开放模型之列——与月之暗面 2.8 万亿参数、定于 7 月 27 日放出权重的 Kimi K3 并肩，并领先于预计本周晚些时候发布正式版的 DeepSeek V4。

为何重要

中国头部实验室如今以数周为间隔接连推出前沿规模模型，且越来越多地搭配开放权重承诺——这一策略让其架构在全球开发者生态中扩散，同时把国内芯片供给无法消化的推理成本转嫁出去。对企业买家而言，一款可信的准前沿模型以十分之一的价格入场，将进一步加剧全行业 API 价格的下行压力。对华盛顿而言，中国公司在多年芯片出口管制之下仍能拿出参数超两万亿的系统，势必给"管制究竟有没有拖慢中国前沿进展"这场本已激烈的争论再添一把火。
